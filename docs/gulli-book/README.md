# Gulli book — per-chapter extracted text

Chapter-by-chapter plain-text extract of Antonio Gulli's *Agentic Design Patterns* (ISBN-less pre-print, 424 pages, extracted from [`docs/Agentic_Design_Patterns.pdf`](../Agentic_Design_Patterns.pdf) via `pdftotext -layout` on 2026-04-23). Files are `.md` so the wiki compiler and RAG embedder can index them.

**Use this directory** when you need the verbatim book text for a pattern. For the platform's implementation status of each pattern, see [`wiki/patterns/`](../../wiki/patterns/) instead. For a short conceptual overview of all 21 patterns, see [`docs/gulli-book-overview.md`](../gulli-book-overview.md).

Files in this directory are extracted text — not hand-edited. Tables, code blocks, and figures may have formatting artefacts from PDF extraction. For pristine reading, open the source PDF.
